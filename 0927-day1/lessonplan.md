# Day 1: Python Fundamentals

## Topics Covered

### 1. Print Output
- Using `print()` to display text
- Escape characters: `\n` (newline), `\t` (tab)
- Single vs double quotes
- Triple quotes for multi-line strings

### 2. Variables & Data Types
- **`str`** - strings (text)
- **`int`** - integers (whole numbers)
- **`float`** - floating-point (decimal numbers)
- **`bool`** - booleans (`True`/`False`)
- **`list`** - ordered collections
- Using `type()` to check data types

### 3. User Input
- `input()` always returns a string
- Converting input to other types

### 4. Type Casting
- `int()` - convert to integer
- `str()` - convert to string
- `float()` - convert to float

### 5. Arithmetic Operators
| Operator | Description | Example |
|----------|-------------|---------|
| `+` | Addition | `3 + 4` → `7` |
| `-` | Subtraction | `3 - 4` → `-1` |
| `*` | Multiplication | `3 * 4` → `12` |
| `/` | Division | `3 / 4` → `0.75` |
| `//` | Floor division | `9 // 4` → `2` |
| `%` | Modulus (remainder) | `7 % 4` → `3` |
| `**` | Exponentiation | `3 ** 4` → `81` |

### 6. F-Strings (Formatted Strings)
```python
name = "Michael"
print(f"Hello, {name}!")
```

### 7. Common Errors
- **NameError** - variable doesn't exist
- **SyntaxError** - typo or syntax mistake
- **TypeError** - mismatch in variable types
- **Logic Error** - code runs but gives wrong result
